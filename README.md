# Tech Challenge 5 - Passos Mágicos

Projeto desenvolvido para o Tech Challenge 5 com foco em análise de dados educacionais, identificação de padrões de desempenho e construção de um modelo preditivo de risco de defasagem escolar utilizando Machine Learning.

---

# Objetivo do Projeto

O objetivo deste projeto é analisar os indicadores educacionais fornecidos pela ONG Passos Mágicos e desenvolver uma solução preditiva capaz de identificar alunos com risco de defasagem educacional.

Além da análise exploratória dos dados, também foi construída uma aplicação web utilizando Streamlit para disponibilizar os resultados e permitir previsões em tempo real.

---

# Tecnologias Utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-Learn
- Streamlit
- Random Forest Classifier
